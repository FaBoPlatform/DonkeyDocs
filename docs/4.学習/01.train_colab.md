## 学習済みデータをDonwload

Colabから学習結果をダウンロードします。Colabの下記コードを実行すると'mypilot.h5'が自動ダウンロードされます。(少々時間がかかります。)

```console
from google.colab import files

files.download('/content/mycar/models/mypilot.h5')
```
