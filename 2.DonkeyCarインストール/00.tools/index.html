使用ツール

使用ツール

機能 アプリ プラットフォーム 用途
SSH Secure Shell Chrome App Raspberry Pi3に接続し、DonkeyCarでコマンドを実装するために使用します。SSH通信が可能なツールならどれでも使用可能です。
SFTP Secure Shell Chrome App データ・セット(datas.zip)をRaspberry Pi3からPCに転送する用途で使います。また、学習済みモデル(mypilot)をPCからRaspberry Pi3に転送するために使います。
Text Editor Text Chrome App TextEditorです。こちらも手持ちのTextEditorがあったら、そちらをお使いください。

使用ツール

SSH, SFTP, TextEditorは普段使っているツールがある場合は、そちらを使用してください。

Chromeアプリのインストール

Secure Shell をクリックし、Chrome Storeからインストールします。

Text をクリックし、Chrome Storeからインストールします。

Chromeアプリの起動

ChromeのURL欄にchrome://appsと入力します。
