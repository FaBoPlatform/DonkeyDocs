    RaspPi3用のイメージ作成

    Imageの作成

    RaspPi3B,B+用SDカードのイメージ https://drive.google.com/open?id=1vr4nEXLEh4xByKAXik8KhK3o-XWgo2fQ よりRaspberryPi 3用のイメージをダウンロードしてくる。

    donkey_2.5.0_pi3.zipを解凍してできた、donkey_2.5.0_pi3.img をSDカードに焼き込む。

    焼き込む方法はこれを参照(OSXの場合) http://docs.fabo.io/fabo/rasppi/dev/pi/install_pixel.html

    Windowsの場合は、Ethcerを使う https://www.balena.io/etcher/

    Donkey関連アプリのインストール

    RaspPi3にHDMIケーブルでDisplayを接続し、RaspberryPiにログインする。

    項目 内容
    ID pi
    PASS raspberry

    最初にWifiの設定をおこなう。

    1
2
    $ sudo chmod 606 /etc/wpa_supplicant/wpa_supplicant.conf
$ sudo sh -c "wpa_passphrase '接続先のSSID' '接続先のパスワード' >> /etc/wpa_supplicant/wpa_supplicant.conf"

    VIMで開き、パスワードの平文を消しておく。(#pskの行)

    1
    $ sudo vim /etc/wpa_supplicant/wpa_supplicant.conf

    1
2
3
4
5
    network={
    ssid="<your network name>"
    #psk="<your password>"
    psk=a7119789798a987a987a98a97a987a98a79a7a987a
}

    Permissionを戻しておく

    1
    $ sudo chmod 600 /etc/wpa_supplicant/wpa_supplicant.conf

    同じ名前のHostnameにあらないように、Hostnameを変更する。

    1
    $ sudo vim /etc/hostname

    /etc/hostname

    1
    donkeypi-akira

    1
    $ sudo vim /etc/hosts

    /etc/hosts

    1
    donkeypi-akira

    ここまででいったん、Rebootする

    1
    $ sudo reboot

    これで、先程つけたHostnameであるdonkeypi-akiraで発見できるようになる。

    1
    $ ssh pi@donkeypi-akira.local

    Host名は、名前をかぶらないように気をつける。
