教師データの作成

DonkeyCarを起動します。

JoyStick

1
2
$ cd ~/mycar
$ python manage.py drive --js

教師データは、~/mycar/dara以下に保存されます。

1
2
3
cd ~/mycar/data
ls
tub_01_19-01-26

CTRL+cで終了します。

教師データの圧縮

データを圧縮し、1つのファイル(ZIP形式)にしてPCに転送します。

RaspberryPi3にZIPをインストールします。

1
sudo apt-get install zip

ZIPでフォルダを圧縮します。

1
2
cd ~/mycar/data
zip -r datas.zip tub_01_19-01-26

データの転送

Cyberduck をインストールします。

RaspberryPi3からPCに、datas.zipを転送します。
